# Revendo

Revendo est une application que j'ai développée pour m'aider à gérer ma micro-entreprise de
revente : suivi du chiffre d'affaires, stock, restock et organisation administrative. Le
projet est actuellement en BETA, utilisé au quotidien et amélioré progressivement. Pour des
raisons de confidentialité, la version publique utilise uniquement des données fictives. Ce
projet met en avant mes compétences en développement d'application, structuration
d'interface, logique métier et gestion de données locales.

**Statut :** BETA — usage quotidien, amélioration continue
**Technologies :** React · TypeScript · Electron · SQLite · Vite · Tailwind CSS
**Démo web (données fictives) :** https://revendo.vercel.app
**Code :** https://github.com/ISURYDEV/revendo
